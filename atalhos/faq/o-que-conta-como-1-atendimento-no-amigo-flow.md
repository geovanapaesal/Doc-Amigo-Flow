# O que conta como 1 Atendimento no Amigo Flow

Um atendimento é **uma conversa iniciada** pela clínica ou pelo paciente com o agente Flow.\
Cada nova conversa aberta é contabilizada como **1 atendimento**, seja reativa (paciente chama) ou proativa (a clínica envia).

#### Disparos Automáticos

Fluxos automáticos, como:

* Confirmação de consulta,
* Mensagem de falta.

{% hint style="info" %}
**Cada envio conta como 1 atendimento.**\
Se o paciente responder, a conversa continua **na mesma janela de atendimento**, sem custo adicional.
{% endhint %}

#### Envios em Massa

Campanhas e comunicações enviadas para vários pacientes (ex: lembretes, campanhas de vacinação, feriados).

{% hint style="info" %}
&#x20;**Cada paciente alcançado conta como 1 atendimento**&#x20;

Se o paciente responder, a conversa continua **na mesma janela de atendimento**, sem custo adicional. Ex: Envio uma campanha para 3 pessoas, 2 me respondem e uma não responde. Terei gasto 3 atendimentos no total.&#x20;
{% endhint %}

#### Início pela Clínica

Quando a recepção inicia uma conversa com o paciente.

{% hint style="info" %}
**Cada conversa iniciada conta como 1 atendimento.**
{% endhint %}

{% hint style="warning" %}
É possível enviar **apenas um modelo de marketing** por atendimento. Caso a clínica tenha iniciado uma conversa com um paciente através de um modelo de marketing e deseje enviar outro, é necessário finalizar o atendimento e abrir um novo.
{% endhint %}

#### Início pelo Paciente

Quando o paciente envia uma mensagem espontaneamente (para agendar, reagendar, tirar dúvidas etc.).

{% hint style="info" %}
**Cada nova conversa iniciada conta como 1 atendimento.**
{% endhint %}

#### Como o atendimento é finalizado

* Automaticamente pelo agente (IA);
* Manualmente pelo atendente;
